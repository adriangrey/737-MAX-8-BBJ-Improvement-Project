# Roadmap

## v0.1.0-alpha

- Increase BBJ fuel capacity to approximately 69,500 lb.
- Use a clean MSFS 2024 modular merge patch.
- Target only the default 737 MAX 8 BBJ preset.

- Research BBJ MAX 8 weight limits.
- Review MTOW, MLW, MZFW, and operating empty weight.

## v0.4.0-beta

- Add SimBrief airframe profile.
- Add basic usage documentation.
- Add screenshots.

## v1.0.0

- First public release candidate.
- Finalize installation instructions.
- Document compatibility and known limitations.
